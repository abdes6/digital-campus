# 浙江树人学院·数字孪生校园系统 — 项目规范文档

## 1. 项目概述

本项目是一个基于 Three.js 的数字孪生校园交互系统，以浙江树人学院拱宸桥校区为原型，在浏览器中实现三维校园场景的可视化与交互。

- **课程**：计算机图形学（期中考核）
- **技术栈**：Three.js r179 · @tweenjs/tween.js · lil-gui · 原生 ES Modules
- **运行方式**：需通过 HTTP 服务器访问（不支持 `file://`）

```bash
python -m http.server 8080
# 访问 http://localhost:8080/three.js/project/index.html
```

```
Z+ = 南   Z- = 北

```

## 5. 模块说明

### 5.1 scene.js

初始化 `THREE.Scene`，包含：

- 天空背景色 + 雾效（`Fog`，80–300 单位）
- 环境光（`AmbientLight`，强度 0.6）
- 主方向光（`DirectionalLight`，模拟太阳，开启阴影，shadowMap 2048×2048）
- 补光（`DirectionalLight`，强度 0.4）
- 地面（400×400 `PlaneGeometry`，绿色 `MeshLambertMaterial`）

### 5.2 camera.js

预设三个视角，切换时使用 `requestAnimationFrame` + easeInOut 插值动画（1200ms）：

| 视角 | 位置 | 朝向 |
|------|------|------|
| 透视视角（默认） | (80, 60, 80) | (0, 0, 0) |
| 鸟瞰视角 | (0, 180, 0.1) | (0, 0, 0) |
| 正面视角 | (0, 30, 120) | (0, 10, 0) |

鸟瞰切换时临时解除极角限制（`maxPolarAngle = Math.PI`）。

### 5.3 controls.js

`OrbitControls` 配置：

- 阻尼：`dampingFactor = 0.06`
- 缩放范围：10–300 单位
- 最大仰角：`Math.PI / 2 - 0.02`（防止穿地）

### 5.5 roads.js

| 道路 | 说明 |

| 湖州街 | 西侧纵向边界，宽 10，X=-115 |
| 日积街 | 东侧纵向边界，宽 10，X=115 |
| 校内南北主干道 | 宽 10，X=0，Z=65 |
| 东西环路（南） | 宽 8，Z=40 |
| 东西环路（北） | 宽 8，Z=-40 |
| 北区南北主路 | 宽 7，X=±75 |
| 宿舍区横向连接路 | 宽 6，X=-90，Z=-35 |

### 5.6 vegetation.js

植被类型：
- **树木**（`createTree`）：三层锥形树冠 + 圆柱树干，高度可配置
- **灌木**（`createBush`）：4 个随机偏移球体组合

植被分布区域：树人街行道树、校内主干道行道树、中心绿轴散树、东西环路行道树、湖州街/日积街边界树、操场周围灌木、宿舍区灌木、教学楼前广场灌木、北部科研区散树。

### 5.7 animation.js

- **`startPatrolAnimation`**：4 段 TWEEN 链式动画，总时长约 12s，经过校园四个方位
- **`pulseObject`**：点击建筑时触发缩放脉冲（scale 1→1.15→1，400ms yoyo）

### 5.8 infoPanel.js

点击建筑/道路/植被时显示右侧信息面板，内容来自对象的 `userData`：
- `userData.name`：面板标题
- `userData.info`：键值对表格（建筑面积、楼层、建成年份、简介等）
- 道路/植被无 `info` 时显示默认描述

## 6. 主逻辑（index.js）

### 6.1 场景组装顺序

```
initScene() → initCamera() → WebGLRenderer → initControls()
→ createBuildings()  [async, 含 GLB 加载]
→ createRoads()
→ createVegetation()
→ 收集所有 Mesh 到 clickables[]
```

### 6.2 图层控制

三个图层（buildings / roads / vegetation）通过左侧复选框控制 `object.visible`。

### 6.3 点击拾取

`Raycaster` 对 `clickables`（所有叶子 Mesh）做射线检测：
1. 向上遍历找到场景直接子节点（建筑根节点）
2. 同步 GUI 面板的位置/旋转滑块
3. 调用 `showInfo` 显示信息面板
4. 触发 `pulseObject` 脉冲动画

### 6.4 建筑位置调整 GUI（lil-gui）

右上角可折叠面板，点击建筑后自动展开并同步当前值：

| 控件 | 范围 | 步进 |
|------|------|------|
| X 位置 | -200 ~ 200 | 0.5 |
| Y 位置 | -10 ~ 50 | 0.5 |
| Z 位置 | -200 ~ 200 | 0.5 |
| 旋转 Y (°) | -180 ~ 180 | 1 |
| 打印坐标 | — | 输出到控制台 |

## 8. 渲染配置

- 抗锯齿：开启（`antialias: true`）
- 像素比：`window.devicePixelRatio`
- 阴影：`PCFSoftShadowMap`
- 动画循环：`requestAnimationFrame` + `TWEEN.update()` + `controls.update()`
