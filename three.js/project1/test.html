星形/齿轮形状生成器

理解代码如何创建星形和齿轮形状

原始代码分析

for(let i = 0; i < points * 2; i++){
  const angle = i * Math.PI / points;
  const r = i % 2 === 0 ? radius : radius / 2;
  const x = r * Math.cos(angle);
  const y = r * Math.sin(angle);
  if(i === 0){
    shape.moveTo(x, y);
  }
  else shape.lineTo(x, y);
}

代码工作原理：

1
循环控制： 循环 points * 2 次，为每个"点"创建两个顶点（一个在外圈，一个在内圈）
2
角度计算： angle = i * Math.PI / points 将圆分成 points 份，但循环两次所以实际创建 points * 2 个点
3
半径交替： 使用 i % 2 === 0 判断，偶数索引使用外半径，奇数索引使用内半径（外半径的一半）
4
坐标计算： 使用极坐标公式 x = r * cos(angle), y = r * sin(angle) 计算每个点的位置
5
路径绘制： 第一个点使用 moveTo()，后续点使用 lineTo() 连接

代码变体与改进

// 变体1: 完整的圆形（2π弧度）
for(let i = 0; i <= points * 2; i++){
  const angle = (i * Math.PI * 2) / (points * 2);
  const r = i % 2 === 0 ? radius : radius * innerRatio;
  const x = r * Math.cos(angle);
  const y = r * Math.sin(angle);
  if(i === 0) shape.moveTo(x, y);
  else shape.lineTo(x, y);
}
shape.closePath(); // 闭合路径
// 变体2: 使用数组存储点，更清晰
const vertices = [];
for(let i = 0; i < points * 2; i++){
  const angle = (i * Math.PI * 2) / (points * 2);
  const r = i % 2 === 0 ? radius : radius * innerRatio;
  vertices.push({
    x: r * Math.cos(angle),
    y: r * Math.sin(angle)
  });
}
// 然后绘制所有点
shape.moveTo(vertices[0].x, vertices[0].y);
for(let i = 1; i < vertices.length; i++){
  shape.lineTo(vertices[i].x, vertices[i].y);
}
shape.closePath();
